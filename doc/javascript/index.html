Bright In Browser

Bright consist of both - server and client side parts. Though it's not necessary to use both. In general your application can run with Ruby server side and Bright client side. Or Bright server side and Backbone client side.

Typical HTML templates

<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="utf-8">
  <title>Site Title</title>
  <meta name="description" content="">
  <meta name="author" content="">
  <meta name="viewport" content="width=device-width, initial-scale=1.0, minimum-scale=1.0, maximum-scale=1.0">
  
  <!--[if lt IE 9]>
    <script src="http://html5shim.googlecode.com/svn/trunk/html5.js"></script>
  <![endif]-->

  <link href="/bright/css/bright.css" rel="stylesheet">
  <script src="/bright/3rdparty/jquery/jquery-1.7.1.min.js"></script>    
  <script src="/bright/js/bright.js"></script>
</head>
